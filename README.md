### 2. 股票市场数据（可获取项，准备下载）

基于 `可以得到的股市数据/` 截图，当前优先级建议如下：

- 必须：`日个股回报率文件`、`日历文件`、`公司文件`
- 建议：`股票历史日行情信息表(后复权)`、`指数文件`、`无风险利率文件`
- 可选增强：`个股走势特征表`、`分配文件`、`股本变动文件`

### 3. 数据主键与时间对齐约定

- 公司主键：`Stkcd`（或 `Symbol` 映射到 `Stkcd`）
- 时间主键：财务类按 `Accper/Reptdt`，事件类按公告日/发生日，交易类按交易日
- 初始样本范围：A股上市公司（后续可按研究需要扩展）
